# 🐍 Fundamentos de Flask con Python

Este repositorio contiene las bases fundamentales del desarrollo web con **Flask**. Fue uno de mis primeros proyectos donde profundicé en la arquitectura de un servidor web y el motor de plantillas Jinja2.

---

## 🚀 Conceptos Aplicados
* **Middleware Básico:** Manejo de funciones `before_request` y `after_request` para el ciclo de vida de la petición.
* **Rutas Dinámicas:** Paso de parámetros por URL (Nombre y Edad).
* **Herencia de Plantillas:** Uso de un `layout.html` base para reutilizar código en múltiples vistas.
* **Manejo de Errores:** Personalización de la página 404 con redireccionamiento inteligente.
* **Query Strings:** Captura y procesamiento de parámetros en la URL.

---

## 🛠️ Tecnologías
* **Backend:** Python / Flask
* **Frontend:** HTML5 / CSS3 (Diseño con herencia)
* **Lógica:** Jinja2
